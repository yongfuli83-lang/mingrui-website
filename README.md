## 部署步骤

### 1. 上传到 GitHub

1. 访问 https://github.com/yongfuli83-lang/mingrui-website
2. 点击 **Add file** → **Upload files**
3. 上传本文件夹所有内容
4. 点击 **Commit changes**

### 2. 在 Vercel 部署

1. 访问 https://vercel.com/new
2. 点击 **Import Git Repository**
3. 选择 **yongfuli83-lang/mingrui-website**
4. 点击 **Deploy**

### 3. 绑定域名

1. 在 Vercel Dashboard 进入项目
2. 点击 **Settings** → **Domains**
3. 添加 `mingruixc.com`
4. 按提示在阿里云配置 DNS
